Why the New Category & Product Aren’t Showing Most Likely Causes: Website Data Source Not Updated: Your website may not be fetching the latest version of the Google Sheet, or there’s a caching/delay issue. Sheet Publishing: If you’re using Google Sheets as a live data source, the sheet might not be published to the web or the changes aren’t reflected in the published version. Troubleshooting Steps Check How Your Website Gets Data Is the website directly fetching from the Google Sheet? Or do you export the sheet to a file (CSV/JSON) and upload it to the repo? If Using Google Sheets API or published CSV/JSON: Make sure the sheet is published to the web. Open the published link directly in your browser to see if the new product is present. If Using Static Data (File in Repo): Update the exported data file (CSV/JSON) with the new sheet data. Commit & push changes to the repo. Wait for GitHub Pages to redeploy (can take a few minutes). Check Filtering in Code: Your site’s code might only display products under certain categories (e.g., only “Tents”). Ensure the code is set to display all categories, not just hardcoded ones. Clear Browser Cache: Sometimes the site may be cached. Try a hard refresh (Ctrl+F5). What to Check Next How is your site pulling in the sheet data? (Live fetch or static file?)
